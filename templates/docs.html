Documentación

Las APIs ofrecidas en el sitio web funcionan como endpoints que pueden ser consumidos a través del método GET desde cualquier otro sitio web mediante el uso de métodos convencionales en Javascript como: fetch API, librería axios, ajax (jQuery), XMLhttpRequest. La estructura de datos, y por ende, la presentación de la información, podría presentar algunas variaciones de acurerdo a cada API en particular, por lo cual la forma como se utilicen y presenten los datos en una interfaz de usuario queda a uso y discreción del desarrollador.


A continuación se presenta un ejemplo básico con Javascript para consumir y presentar en una interfaz gráfica una de las APIs disponibles en el sitio web: 

              
                function getApiData() {

                  // Div where data is displayed
                  const fetchingDiv = document.querySelector("#fetching-div");
                  const endpoint = "The API endpoint";
                
                  fetch(endpoint)
                    .then((response) => response.json())
                    .then((data) => showData(data))
                    .catch((error) => console.log(error));

                  // Function to show data on UI
                  function showData(data) {

                    data.forEach((character) => {
                      const card = document.createRange().createContextualFragment(
                        ` 
                        Your HTML here
                        `
                      );

                      fetchingDiv.appendChild(card);
                    });
                  }
                }
